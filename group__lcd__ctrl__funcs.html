◆ lcd_create()

lcd_t* lcd_create ( lcd_config_t  config,
const int *  control_pins,
const int *  data_pins 
)

Create LCD object.

The config flags might be combined using the OR operator.

The control_pins must be provided in the following sequence: RS, EN, RW (RW only if LCD_RW_SUPPORT is defined).

The data_pins must be provided in the following sequence:

  • D0, D1, D2, D3, D4, D5, D6, D7 for 8bit; or
  • D4, D5, D6, D7 for 4bit

Example:

// {RS_port, RS_pin, EN_port, EN_pin}
const int control_pins[] = {0, 1, 0, 2};
// {D4_port, D4_pin, D5_port, D5_pin, D6_port, D6_pin, D7_port, D7_pin}
const int data_pins[] = {0, 3, 0, 4, 0, 5, 0, 6};
// create lcd object
lcd_t *lcd = lcd_create(LCD_4BIT | LCD_2LINE, control_pins, data_pins);
Parameters
[in]configthe configuration of the LCD
[in]control_pinsinteger array with the control pins
[in]data_pinsinteger array with the data pins
Returns
pointer to the LCD object or NULL if no more instances are available
